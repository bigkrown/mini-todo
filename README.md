# Todo App with localStorage Persistence

A simple, elegant todo application built with JavaScript and React, featuring automatic localStorage persistence. Keep track of your tasks across browser sessions!



## Live Demo

https://shmini-todo-app.netlify.app/

## Features

- **Add, delete, and complete todos** - Full CRUD operations for your tasks
- **Automatic localStorage persistence** - Todos are saved automatically and persist across browser sessions and page reloads
- **Cross-tab synchronization** - Changes made in one tab are reflected in all open tabs
- **Clean, intuitive interface** - Simple and focused design
- **Responsive design** - Works on desktop, tablet, and mobile
- **Real-time updates** - UI updates instantly as you make changes
- **Todo counter** - Always know how many tasks you have

## Technologies Used

- **JavaScript (ES6+)** - Modern JavaScript with classes and modules
- **React** - For building the user interface
- **localStorage API** - For data persistence
- **CSS3** - For styling

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/bigkrown/mini-todo.git
   cd mini-todo

2. **Install dependencies**
npm install

3. **Start the development server**
npm start

4. **Open your browser**
Navigate to http://localhost:5173



**Usage**
Adding a Todo
Type your task in the input field and press Enter or click the Add button.

Completing a Todo
Click the checkbox next to any todo to mark it as complete. The text will be crossed out.

Deleting a Todo
Click the "delete" button next to any todo to remove it permanently.

Todo Counter
The counter next to "Todo List" shows you how many active todos you have.
